## 为什么选择 Astro

Astro 很适合内容型站点，因为它强调静态输出与轻量加载。

### 写作体验

使用 Markdown 管理文章，可以把注意力更多放在内容上。

### 页面性能

默认输出更轻，对博客这类站点非常友好。

## 代码示例

```ts
const message = 'Hello Astro';
console.log(message);
```
